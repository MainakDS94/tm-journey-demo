# Trademark Lifecycle — Questel Product Mapping

This repository contains an **interactive walkthrough of the trademark lifecycle**, illustrating how Questel products support each phase — from name ideation and clearance through enforcement and maintenance.

The demo is designed to:
- Visualise typical trademark workflows
- Highlight roles (IP managers, paralegals, legal teams, law firms)
- Show how Questel solutions fit into real operational journeys

**Purpose**
- Internal demonstration and discussion
- Product understanding and storytelling

**Disclaimer**
This is a conceptual demo created for illustration purposes only.  
It is **not a production system** and should not be used for commercial or operational decision-making.
